TROUBLESHOOTING QUICK REFERENCE
================================================================================

Issue: "Connection refused"
  Solution: Start backend server
  Command: npm run backend:dev

Issue: "Authentication failed"
  Solution: Seed database with test users
  Command: npm run prisma:seed

Issue: "Categories not found"
  Solution: Database may need seeding
  Command: npm run prisma:seed

Issue: "Request timeout"
  Solution: Backend may be slow or busy
  Action: Check backend logs, increase TIMEOUT in script

Issue: "Database locked"
  Solution: Multiple backend instances running
  Action: Kill all backends, restart one

For more help:
  - QUICK_TEST.md → Troubleshooting section
  - API_TEST_GUIDE.md → Troubleshooting section
  - Check backend logs: npm run backend:dev
